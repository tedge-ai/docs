# Timeframes

## All Available Timeframes

tedge.ai supports all 21 MetaTrader 5 timeframes:

| Category | Timeframes |
|----------|-----------|
| Minutes | M1, M2, M3, M4, M5, M6, M10, M12, M15, M20, M30 |
| Hours | H1, H2, H3, H4, H6, H8, H12 |
| Higher | D1 (Daily), W1 (Weekly), MN (Monthly) |

## Favorites

Customize which timeframes appear in the quick access bar:
- **Desktop** — Click the chevron next to the TF pills, then star your favorites
- **Mobile** — Tap "...", then long-press a TF to toggle favorite

## Data Sources

- **MT5 Bridge connected** — All TFs natively from your broker
- **Twelve Data fallback** — Natively supports M1, M5, M15, M30, H1, H2, H4, D1, W1, MN. Other TFs (M2, M3, H6, etc.) are automatically calculated by aggregating from the nearest base TF.
