# Charts

Professional charting powered by TradingView Lightweight Charts.

## Features

- **21 timeframes** — From M1 to Monthly, including M2, M3, H6, H12
- **Drawing tools** — Horizontal lines, trendlines, channels, Fibonacci, arrows, positions
- **Indicators** — EMA, SMA, Bollinger Bands, RSI, ADX (AI-configurable)
- **Infinite scroll** — Load historical data by scrolling left
- **AI overlay** — Zones and levels from analysis appear directly on the chart
- **Real-time updates** — Live price feed via MT5 Bridge or Twelve Data
