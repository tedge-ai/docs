# AI Analysis

Edge AI analyzes market structure to identify trading opportunities. The agent examines multiple timeframes, identifies key zones, and proposes trade setups.

## How It Works

1. **Select a strategy** — Choose from 6 pre-defined strategies or create a custom one
2. **Click Analyze** — The AI agent starts a multi-step analysis
3. **Phase 1** — Structure reading (H4 → H1 → M15), zone identification
4. **Phase 2** — Opportunity construction, entry/SL/TP calculation
5. **Results** — Zones appear on chart, opportunities in the analysis panel

## Cost

Each analysis consumes AI credits (~0.30-0.70 depending on depth). Credits are included in your plan.
