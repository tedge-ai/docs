# Push Notifications

## Setup

1. When you first visit the platform, a bell icon appears in the topbar
2. Click it to enable push notifications
3. Allow notifications in your browser
4. You're set!

## PWA (Mobile)

For the best mobile experience:
1. Open the platform in Safari (iOS) or Chrome (Android)
2. Add to Home Screen
3. Enable notifications when prompted
4. Receive alerts even when the app is in the background

## Telegram

Link your Telegram account for an additional notification channel:
1. Go to **Profile** page
2. Click **Link Telegram**
3. Click the generated link to open @tedgeaibot in Telegram
4. Send /start to complete the pairing

### Telegram Commands

| Command | Description |
|---------|-------------|
| `/positions` | List your open demo positions |
| `/stats` | Show your trading statistics |
| `/analyse BTCUSD` | Run an AI analysis on a symbol |
