# Price Alerts

## Creating an Alert

1. Open the **Alertes** panel
2. Click **+ New Alert**
3. Select the symbol and set the target price
4. Choose the condition (crosses above / crosses below)
5. Save

## Alert Limits

| Plan | Active Alerts |
|------|--------------|
| Free | 3 |
| Basic | 20 |
| Advanced | 100 |
| Premium | Unlimited |

## When Triggered

When price reaches your alert level:
- Push notification on your device (if enabled)
- Telegram message (if linked)
- Alert marked as triggered in the panel
