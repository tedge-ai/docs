# Demo Mode

## Virtual Account

Every user gets a demo account with $10,000 virtual balance. Trades are simulated using real market prices.

## How It Works

- Trades execute at the current market price
- P&L is calculated using real-time prices
- SL and TP trigger automatically when price reaches the level
- MFE (Maximum Favorable Excursion) and MAE (Maximum Adverse Excursion) are tracked

## Resetting

Contact support to reset your demo balance to the initial amount.
