# Trade History

## Viewing History

Navigate to the **Trades** page for full history, or expand the **Historique** section in the Trades panel.

## Columns

| Column | Description |
|--------|-------------|
| Date | When the trade was opened |
| Symbol | Instrument traded |
| Direction | BUY or SELL |
| Lots | Position size |
| Entry / Exit | Prices |
| P&L | Profit or loss in dollars |
| R:R | Risk-to-reward ratio achieved |
| MFE | Best unrealized P&L during the trade |
| MAE | Worst unrealized P&L during the trade |
| Duration | How long the position was held |
| AI Score | AI Coach rating (1-10) |

## Statistics

The Trades page includes performance statistics:
- Win rate, average P&L, total P&L
- Trading calendar with daily results
- Equity curve
