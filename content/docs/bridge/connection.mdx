# Connecting

## Pairing Your Account

1. Start EdgeBridge on your Windows machine
2. On the Tedge platform, go to the **Bridge** page
3. Enter the pairing code shown in EdgeBridge
4. Click **Pair**
5. The connection status turns green

## Connection Status

Look for the connection indicator in the topbar:
- **Green dot** — EdgeBridge connected, live data active
- **Gold dot** — Twelve Data fallback (no bridge)
- **Purple dot** — Demo mode

## Auto-Reconnect

EdgeBridge automatically reconnects if the connection drops. No manual intervention needed.
