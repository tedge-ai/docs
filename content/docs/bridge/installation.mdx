# Installation

## Download

1. Log in to [app.tedge.ai](https://app.tedge.ai)
2. Navigate to the **Bridge** page
3. Download the EdgeBridge installer for Windows
4. Run the installer and follow the setup wizard

## Setup Wizard

The installer will guide you through:
1. **MT5 Path** — Automatically detects your MetaTrader 5 installation
2. **Pairing** — Generates a pairing code to link with your tedge.ai account
3. **System Tray** — EdgeBridge runs in the background with a system tray icon

## System Requirements

- Windows 10 or later
- MetaTrader 5 installed
- Python runtime (bundled with the installer)
