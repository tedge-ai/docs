# Getting Started

Get up and running with Tedge in a few minutes.

## Quick Start

1. **Create your account** — Sign up at [app.tedge.ai](https://app.tedge.ai/sign-up)
2. **Complete onboarding** — Set your trading style, risk preferences, and favorite symbols
3. **Run your first analysis** — Select a symbol, choose a strategy, and hit Analyze
4. **Review the results** — Explore the AI-detected zones, opportunities, and trade setups

## What You'll Need

- A modern web browser (Chrome, Firefox, Safari, Edge)
- An email address for account creation
- Optionally: MetaTrader 5 for live data via EdgeBridge

## Platform Overview

Tedge is an AI-powered trading intelligence platform that combines:

- **Professional charting** with 21 timeframes, drawing tools, and indicators
- **AI market analysis** using Smart Money Concepts and other strategies
- **Trade management** in demo and live mode
- **Smart alerts** with push notifications
- **AI coaching** with post-trade reviews and scoring
