# Drawing Tools

## Available Tools

Access drawing tools from the pencil icon in the topbar.

### Lines
| Tool | Description |
|------|-------------|
| **Horizontal Line** | Draw a horizontal level at a specific price |
| **Vertical Line** | Mark a specific time on the chart |
| **Trendline** | Connect two points to draw a trend line |
| **Ray** | Draw a line from one point extending infinitely |
| **Arrow** | Draw an arrow between two points |

### Analysis
| Tool | Description |
|------|-------------|
| **Fibonacci** | Fibonacci retracement levels between two points |
| **Rectangle** | Draw a rectangular zone (support/resistance area) |
| **Parallel Channel** | Draw a price channel with parallel lines |
| **Long/Short Position** | Visualize a trade setup with entry, SL, TP |

## How to Draw

1. Click the drawing tool icon in the topbar
2. Select a tool from the dropdown
3. Click on the chart to place the first point
4. Click again for the second point (where applicable)
5. The drawing appears and can be moved or deleted

## Deleting Drawings

- Right-click on a drawing to access the context menu
- Or use "Delete all" at the bottom of the drawing tools dropdown
