# Trading

Tedge supports both demo and live trading with full position management.

## Modes

- **Demo mode** — Paper trading with a virtual $10,000 balance. Perfect for testing strategies risk-free.
- **Live mode** — Real trades via MT5 Bridge (Basic+ plans). Requires EdgeBridge connected.

Toggle between modes using the DEMO/LIVE button in the topbar.

## Features

- Open BUY/SELL positions from the chart or trade panel
- Set SL and TP levels
- Real-time P&L tracking
- Position overlay on chart
- AI Coach review on trade closure
- Full trade history with statistics
